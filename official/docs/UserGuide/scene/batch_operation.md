**2. 课程目标**
- `Alt` + 单击鼠标右键：替换单个方块
- `Ctrl+Z`：撤消之前的操作
- `Ctrl+Y`：恢复之前的操作
- `Ctrl+S`：保存
- `Shift` + `Alt` + 单击鼠标右键：批量替换方块
- `Shift` + 单击鼠标右键：扩展选中的方块直到触碰到另一种方块
- `Shift` + 单击鼠标左键：删除同种类的方块
- `Ctrl` + 单击或拖动鼠标左键：选中方块区域
  - 将选中的方块 `保存` 为 `template模板`（全局模板文件可以在多个世界中共享）
- 加载 `模板` 文件：`E` 键 --> `模板` 标签 --> `全局模板`
- `Tab` 键：人物到正上方一层
- `Shift` + `Tab` 键：人物到正下方一层
- `帮助页` (`F1`)：`Esc` 键 --> `帮助` 菜单 --> `帮助...`
- `Ctrl` + `F` 键：快速搜索世界中的代码方块、模型和告示牌

**步骤2：请在键盘上找到 `Ctrl`, `Alt`, `Shift` 键。它们是可以配合其他键或鼠标使用的特殊键。
按住它们，同时点击另一个键，形成一个复合键。比如，`Ctrl+S`是一个组合键，相当于 `/save` 命令。要执行它，先按下 `Ctrl` 键，然后按 `S` 键。程序员使用大量的复合键来执行预定义的命令。请用下列复合键练习。**


`单击右键` 是点击鼠标的右键。
`拖动右键` 是按住鼠标右键并移动鼠标。 

- `Alt` + `单击右键`：替换单个方块
- `Alt` + `单击左键`：吸取方块材质
- `Ctrl+Z`：取消上次操作。如果你删除或创建了一些方块，你不喜欢它们，你可以按下 `Ctrl+Z` 来取消，可以连续按。
- `Ctrl+Y`：恢复上次操作。
- `Ctrl+S`：保存。和 `/save` 命令功能相同。你可以尝试输入 `/save` 然后按 `Enter` 键来执行相同的操作。
- `Shift` + `Alt` + `单击右键`：批量替换方块
- `Shift` + `单击右键`：扩展选中的方块直到触碰到另一种方块
  - `Shift` + `拖动右键`：扩展选中的方块到所选的区域
- `Shift` + `单击左键`：删除几个同种类的方块
  - `Shift` + `拖动左键`：删除所选区域的方块
- `Ctrl` + `单击或拖动左键`：选中方块区域
  - `Ctrl+C` 复制所选方块，`Ctrl+V` 粘贴到鼠标所在的位置
  - `保存` 所选方块为 `template模板`（全局模板文件可以在多个世界中共享）
- 加载 `模板` 文件：`E` 键 --> `模板` 标签 --> `全局模板`
- `Tab` 键：人物到正上方一层
- `Shift` + `Tab` 键：人物到正下方一层
- `Ctrl` + `F` 键：通过按 `Ctrl` + `F` 键，可以快速检索到你场景中的代码方块、Bmax模型或告示牌。
  - 按下`Ctrl` + `F` 键之后，会出现资源搜索窗口，如下图所示。

收获：当我们使用快捷键来创建3D世界时，我们实际上是在编写一系列命令，并让计算机逐一执行。因此，你输入的每一个有效的快捷键都算作你写的新代码行。你将在每一堂课中编写50行代码。当你达到5000行代码时，你可以称自己为年轻的程序员了。
