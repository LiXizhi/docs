

# 多人在线编辑3D世界

- 通过命令，Paracraft客户端可以随时变成服务器，允许外部用户接入一起创作。 
- 多人联网支持基本的方块和人物的同步，可多人一起编辑同一个世界。
- 支持通过NPL tunnel服务器链接，让没有固定IP地址的客户端能成为互联网服务器。
