# 关于课程智能模组

<details style="background-color:white">
  <summary>什么是智能模组？</summary><p>

### 设计背景
支持低代码开发的模组。用资源商城的方式快速构建可交互应用（虚拟+物理）。
用户：面向跨学科课件的开发（中小学老师）和学员(3-5年级)。

可以节省开发时间，避免重复劳动，并保持界面的一致性和易用性。通过使用这些模板，可以帮助用户快速构建软件的交互、操作逻辑和界面美观。此外，低代码模块还具有自定义图块编程的功能，作为主要的人机接口，大多数情况下只需要修改参数，而不需要进行新的编程。


### 目标
对有一定基础的成人0学习成本拿来就会用， 对3-5年级自学后可举一反三。
自定义图块编程作为主要人机接口，大多数情况只需修改参数，不用新写。
可用于师训和C端长期课，提升作品质量，扩大应用场景，增加开发效率。
低代码模块是一种支持低代码开发的模组，它能够帮助用户以更高效的方式构建可交互应用，无论是虚拟应用还是物理设备，都可以通过低代码模块来实现。这种模组的背景是为了满足当今跨学科课件开发的需求，尤其是针对中小学老师和3-5年级的学员。


总的来说，低代码模块是一种非常实用的工具，它可以帮助教育者和设计师快速构建出成熟、完整的课件或作品，并为跨学科课件的开发提供了一种新的方式。通过使用这种模组，用户可以更快速地构建出符合自己需求的应用，并以更高的效率来完成任务。
  

</p></details>

<font color="#994C00">

我们从资源商城中选择植物生长模板并在场景中进行使用，自动生成的种子商人NPC，会有不同的水壶和种子贩售，我们可以在商人中选择自己喜欢的种子进行种植。还可以移动和复制该NPC进行商品定制，把自定义的种子放到商人里卖。

  **使用简易说明**

除了默认的种植土地（方块ID为13），用户还可以自定义土地，例如将种植土地方块改为13、55、3号，在这三种方块上都可以进行种植。

如果商店中的所有植物都不需要售卖，用户可以自定义种子商人售卖的物品，例如复制一些商人NPC并只售卖特定的物品。

总的来说，植物生长模板是一种非常灵活和强大的工具，可以帮助教育者和设计师快速生成植物并自定义多种参数，为跨学科课件的开发提供了一种新的方式。通过使用这种模板，用户可以更快速地构建出符合自己需求的植物，并以更高的效率来完成任务。

智能识别模块目前主要功能为摄像头识别物品以及写数字让小动物做相对应的动作。

摄像头识物是运用摄像头捕获图像并传递到图像识别算法，图像识别算法将捕获的图像转换为数字化信息，以便进行分析和识别。根据算法的识别结果，摄像头可以判断出物品的名称、类型和其他相关信息。我们甚至可以利用该功能，做一些识别物品直接在场景中创建该物品模型等设计。
